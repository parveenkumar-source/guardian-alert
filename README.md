# 🚨 Women Safety SOS System

## 📌 Overview
A quick emergency alert system for women to send SOS with live location in one tap.

## 🎯 Problem
In danger situations, calling for help is difficult and slow.

## 💡 Solution
One-tap SOS sends real-time location and alerts to emergency contacts.

## 🚀 Features
- 🆘 SOS Button  
- 📍 Live Location  
- 📞 Emergency Alerts  
- 🔔 Real-time Notification  

## ⚙️ Setup

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```
